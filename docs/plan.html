System Architecture — Local-First Hybrid

DeepSeek-এর Local-First philosophy + Claude-এর backend scalability combined।

High-Level Flow DP

1. User logs in → Google OAuth 2.0 → Access token stored securely
2. App loads → IndexedDB থেকে local notes render (instant)
3. User edits → Zustand state update → Command history push
4. Canvas change → 15s debounce → IndexedDB save
5. Sync Engine → IndexedDB diff → Drive API update (JSON file)
6. Conflict detected → User notification → Merge/override choice
7. Export → html2canvas → jsPDF বা docx.js → Download

Drive File Structure D

Google Drive/ ├── InkFlow Studio/ # App folder │ ├── Physics Notes/ │ │ ├── meta.json # notebook info, bg, color │ │ ├── page_1.json # canvas elements │ │ └── page_2.json │ └── Daily Journal/ │ └── page_1.json └── user_prefs.json # settings, themes

Canvas Rendering Pipeline CD

Draw: PointerEvent → raw points[] → perfect-freehand smoothing → Konva Line render
Shape detect: pen-up → Douglas-Peucker simplify → $1 Recognizer → 85%+ confidence → auto-snap
OCR mode: stroke complete → MyScript API call → text replace (online) / Tesseract (offline)
Serialize: Konva canvas → JSON (elements array) → IndexedDB → Drive

Security Model DP

Zero-Knowledge (DeepSeek): Notes শুধু user-এর Google Drive-এ। আমাদের server-এ কোনো note data নেই।
HTTPS everywhere: সব API call TLS। PKCE OAuth2 flow।
Token storage: HttpOnly cookies। Browser storage-এ কোনো secret নেই।
GDPR (ChatGPT): Data export/delete on request। Privacy policy। Consent screen।
API keys: শুধু server-side (Next.js API routes)। Client-এ কোনো secret নেই।

Data Schema — Combined Best

DeepSeek-এর JSON schema + Claude-এর PostgreSQL schema combined।

Note JSON Structure (per page) DP

{ "id": "page_uuid", "notebookId": "nb_uuid", "background": { "color": "#ffffff", "pattern": "grid" // blank|lined|grid|dotted }, "elements": [ { "type": "drawing", "strokes": [{ "points": [{x,y,pressure}], "brush": {color, size} }], "recognizedText": "Hello" // OCR result — for search }, { "type": "shape", "shapeType": "rectangle", "position": {"x":100, "y":200}, "size": {"w":150, "h":100}, "style": {"stroke":"#000", "fill":"transparent"} }, { "type": "text", "content": "Note content...", "style": {"fontSize":16, "bold":true, "font":"Noto Sans Bengali"}, "position": {"x":100, "y":300}, "size": {"width":400, "height":200} }, { "type": "image", "src": "drive_file_id_or_base64", "position": {"x":50, "y":50}, "size": {"width":300, "height":200} }, { "type": "audio", // Gemini's Audio Sync feature "src": "drive_audio_id", "timestamps": [{"elementId":"el_001", "time":42.5}] } ], "version": 3, // conflict resolution "updatedAt": "2026-05-06T..." }
