# InkFlow License - Customer Steps

এই file customer-side flow বোঝার জন্য।

## customer যা পাবে
1. App URL
2. Email
3. License Key

Example:
- App URL: https://foylx.vercel.app
- Email: a3kmstudio@gmail.com
- License Key: INKF-U3VN-HR6G-EWXF

## customer কী করবে
1. App খুলবে
2. Email লিখবে
3. License key লিখবে
4. Activate চাপবে
5. Success message দেখবে
6. App unlock হবে

## password লাগবে কি?
- না
- এখনকার system-এ password লাগে না
- শুধু email + key লাগে

## যদি error আসে
1. Email ঠিক আছে কি না দেখবে
2. Key ঠিক আছে কি না দেখবে
3. Internet আছে কি না দেখবে
4. key revoke হয়েছে কি না admin দেখবে

## customer experience কেমন
1. প্রথমবার activation screen আসবে
2. key valid হলে unlock হবে
3. same device-এ পরে cache দিয়ে খুলতে পারে
4. new device হলে আবার activate করতে হতে পারে
